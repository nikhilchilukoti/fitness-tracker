# Fitness Pro | Premium Tracker

A modern, high-performance fitness dashboard built with React, Vite, and Framer Motion.

## Features
- **UI Lite**: Minimalist, clean design focused on clarity.
- **Automated Deployment**: Built and deployed automatically via GitHub Actions.
- **High Performance**: Optimized for fast load times and smooth animations.

## Live Demo
Check out the live app here: [https://nikhilchilukoti.github.io/fitness-tracker/](https://nikhilchilukoti.github.io/fitness-tracker/)

## Quick Start
```bash
npm install
npm run dev
```

## Deployment
This project is configured to deploy to GitHub Pages using GitHub Actions. To deploy:
1. Push to the `main` branch.
2. Ensure repository settings are set to "GitHub Actions" under the Pages tab.
